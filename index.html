    Welcome to the Reina API documentation website! Reina API is an API which mainly focuses on Anime. It doesn't requires an API key.

    Errors

    The Reina API uses the following error codes:

    Error Code Meaning
    400 Bad Request -- Your request is invalid.
    404 Not Found -- The specified endpoint could not be found.
    500 Internal Server Error -- We had a problem with our server. Try again later.
    503 Service Unavailable -- We're temporarily offline for maintenance. Please try again later.

    This endpoint gets a random Waifu/Husbando from MyWaifuList and returns the information of the character.

    GET https://reina-api.vercel.app/api/mwl/random

    This endpoint returns the information of the given slug from MyWaifuList.

    HTTP Request

    GET https://reina-api.vercel.app/api/mwl/character/slug/:slug

    Path Parameters

    Parameter Required Description
    slug true Slug of the waifu from MyWaifuList
